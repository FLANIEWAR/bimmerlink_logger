# bimmerlink_logger

Простой веб-интерфейс для загрузки CSV-логов, просмотра исторических записей и построения мобильных графиков.

## Установка

1. Перейдите в папку проекта:
   ```bash
   cd path/to/bimmerlink_logger
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер:
   ```bash
   npm start
   ```

## Работа через pm2

```bash
pm2 start npm --name bimmerlink_logger -- start
```

## Возможности

- Загрузка CSV-файлов логов
- Сохранение истории всех записей в общую базу SQLite
- Просмотр архива и выбор записи для анализа
- Отображение графиков по выбранным параметрам
- Режимы: на одном графике или раздельно
- Показ максимальных значений по каждому параметру
